# 真实人物样板 01 · 候选选择与网站适配执行日志

> 线程 ID：运行时未提供，使用日期与任务短名作为回退标识。

## 任务契约

- 目标：在名人方案与全面模拟方案中选定一个方向，确定一位候选人，建立可核验的人物资料包，并产出可直接进入甲乙方确认的网站适配与视觉设计方案。
- 用户约束：暂不深度集成真实自媒体账号或官方网站；社交区只保留若干通用模型和占位跳转。短期个性化仅允许内容重构、换皮换色、贴图/UI、现有区域适配和至少一个个人雕像，不新增大场景或高成本玩法。
- 输出：候选比较与选择证据、人物事实包、人生—空间映射、项目/Lab/履历/看板方案、雕像和风格资产清单、实施边界、验收标准、风险与授权说明、可复用工作流。
- 非目标：本轮不把名人形象直接发布上线，不伪造背书，不复制受版权保护的媒体素材，不开发新场景或实时社媒接口。
- 证据边界：人物事实优先使用本人/所属机构、一手采访、官方记录和可靠媒体；推断与设计转译必须明确标注。

## 决策日志

- 选择“方案一：名人方案”作为本轮执行路径。理由：公开事实可被第三方核验，能真实测试资料策展、内容证据和权利边界；全面模拟方案暂作为无法取得真实客户资料时的后备方法。
- 不把社交账号作为人物差异化核心，只保留小红书、B站、公众号、邮箱等通用联系模型；人物魅力主要通过履历、项目、Lab、时间机器、色彩/材质和雕像表达。
- 交付以设计蓝图和可执行清单为本轮终点，不自动改站或公开发布。
- 候选条件性选择韩寒。其内部适配加权分最高（89.8），但权利可控性只有 1.5/5，低于工作流公开生产门槛 3/5；因此仅通过内部研究候选闸门，未通过公开/商业生产闸门。
- 人物命题冻结为内部策展提案《两种速度——在文字、赛道与镜头之间》；Landing、Career、4 个 Projects、6 个 Lab、Circuit、Vehicle、Time Machine 和一座抽象低模雕像均限制在 L0–L2。
